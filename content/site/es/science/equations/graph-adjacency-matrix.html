Matriz de Adyacencia de Grafo

Ecuación

$$A_{ij}=1\ \text{si el nodo }i\text{ conecta con el nodo }j,\quad A_{ij}=0\ \text{en caso contrario}$$

Significado Claro

Una red puede guardarse como una tabla que muestra qué nodos están conectados.

Símbolos

  • \(A_{ij}\): entrada para la conexión del nodo \(i\) al nodo \(j\).
  • 1: conexión presente.
  • 0: conexión ausente.

Cómo Leerla

Una vez codificada así, las operaciones matriciales pueden revelar caminos, hubs, grupos y flujo de influencia.

Significado en Contexto

La matriz significa un registro compacto de qué nodos se conectan con cuáles. Pesos, direcciones o dependencia temporal pueden enriquecer el modelo de grafo.
