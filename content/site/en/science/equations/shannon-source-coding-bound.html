Shannon Source Coding Bound

Equation

$$ H(X)\le L < H(X)+1 $$

Literal Reading

The average code length \(L\) can be made at least the entropy and less than one bit above the entropy.

Physicist's Reading

A physicist would read this as the irreducible average description length of a long classical source.

Symbol Breakdown

\(H(X)\): Shannon entropy of the source.

\(L\): expected number of bits per symbol used by a prefix code.

\(+1\): the finite-symbol overhead from using whole bits in a practical code.

Physical Meaning

Compression is not arbitrary shrinking. It removes statistical redundancy. Entropy is the lower limit for lossless compression when many source symbols are encoded together.
