Shannon Source Coding Bound

Equation

$$ H(X)\le L < H(X)+1 $$

Literal Reading

The average code length \(L\) can be made at least the entropy and less than one bit above the entropy.

Common Reading

Short version: The irreducible average description length of a long classical source.

Symbols

\(H(X)\): Shannon entropy of the source.

\(L\): expected number of bits per symbol used by a prefix code.

\(+1\): the finite-symbol overhead from using whole bits in a practical code.

Meaning of the Equation

The irreducible average description length of a long classical source.
