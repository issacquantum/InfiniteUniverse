Hoare Triple

Equation

$$ \{P\}\;C\;\{Q\} $$

Literal Reading

If precondition \(P\) is true before command \(C\), then postcondition \(Q\) is true after \(C\) finishes.

Common Reading

Short version: A conservation statement for program meaning: the code must transform allowed starting states into guaranteed final states.

Symbols

\(P\): precondition, what must be true before execution.

\(C\): command or program fragment.

\(Q\): postcondition, what must be true afterward.

Meaning of the Equation

A conservation statement for program meaning: the code must transform allowed starting states into guaranteed final states.
