Hoare Triple

Equation

$$ \{P\}\;C\;\{Q\} $$

Literal Reading

open brace P close brace C open brace Q close brace.

Common Reading

A conservation statement for program meaning: the code must transform allowed starting states into guaranteed final states.

Symbols

\(P\): precondition, what must be true before execution.

\(C\): command or program fragment.

\(Q\): postcondition, what must be true afterward.

Meaning of the Equation

The triple is a logic contract for code correctness. It proves that every allowed start state leads to the promised end state, not merely that tests happened to pass.
