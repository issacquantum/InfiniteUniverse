Hoare Triple

Equation

$$ \{P\}\;C\;\{Q\} $$

What the Formula Says

If precondition \(P\) is true before command \(C\), then postcondition \(Q\) is true after \(C\) finishes.

What It Means in Practice

This means a conservation statement for program meaning: the code must transform allowed starting states into guaranteed final states.

Symbols

\(P\): precondition, what must be true before execution.

\(C\): command or program fragment.

\(Q\): postcondition, what must be true afterward.

Equation Meaning

Correctness is not only whether a program ran once. It is whether the program is guaranteed to move from valid inputs to valid outputs under stated assumptions.
