Logistic Map

Equation

$$x_{n+1}=r x_n(1-x_n)$$

Literal Reading

x sub open brace n plus 1 close brace equals r x sub n open parenthesis 1 minus x sub n close parenthesis.

Common Reading

The next value is produced from the current value by a simple nonlinear rule.

Symbols

  • \(x_n\): current value.
  • \(x_{n+1}\): next value.
  • \(r\): control parameter.

Meaning of the Equation

The next state is generated from the current state; simple rules can produce complex behavior.
