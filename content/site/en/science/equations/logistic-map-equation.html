Logistic Map

Equation

$$x_{n+1}=r x_n(1-x_n)$$

Literal Reading

x sub n plus 1 equals r x sub n of 1 minus x sub n.

Common Reading

The next value is produced from the current value by a simple nonlinear rule.

Symbols

  • \(x_n\): current value.
  • \(x_{n+1}\): next value.
  • \(r\): control parameter.

Meaning of the Equation

The logistic map is a simple discrete rule whose behavior can shift from stable to chaotic as the parameter changes. It shows how deterministic equations can generate unpredictability.
