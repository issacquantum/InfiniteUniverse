Graph Adjacency Matrix

Equation

$$A_{ij}=1\ \text{if node }i\text{ connects to node }j,\quad A_{ij}=0\ \text{otherwise}$$

Literal Reading

Read it as: A i j equals one if node i connects to node j, and zero otherwise.

Common Reading

A network can be stored as a table showing which nodes are connected.

Symbols

  • \(A_{ij}\): entry for the connection from node \(i\) to node \(j\).
  • 1: connection present.
  • 0: connection absent.

Meaning of the Equation

This relation turns an algorithmic step into a precise rule. It identifies what is being updated, counted, or bounded, so the process can be checked by its state, cost, error, or learning behavior instead of only by a visual animation.
