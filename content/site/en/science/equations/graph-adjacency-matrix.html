Graph Adjacency Matrix

Equation

$$A_{ij}=1\ \text{if node }i\text{ connects to node }j,\quad A_{ij}=0\ \text{otherwise}$$

Plain Meaning

A network can be stored as a table showing which nodes are connected.

Symbols

  • \(A_{ij}\): entry for the connection from node \(i\) to node \(j\).
  • 1: connection present.
  • 0: connection absent.

How to Read It

Once encoded this way, matrix operations can reveal paths, hubs, clusters, and influence flow.

Meaning in Context

The matrix means a compact record of which nodes connect to which other nodes. Extra weights, directions, or time dependence can enrich the graph model.
