Polynomial Matrix-Vector Transform

Equation

$$ p(A)b $$

Literal Reading

Read it as: y equals p open parenthesis A close parenthesis x.

Common Reading

Apply the polynomial or matrix function \(p\) to matrix \(A\), then apply the result to vector \(b\).

Symbols

\(A\): matrix or linear operator.

\(p\): polynomial or matrix function applied to \(A\).

\(b\): vector acted on by the transformed matrix.

Meaning of the Equation

A matrix transforms an input vector into an output vector.
