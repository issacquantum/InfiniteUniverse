Recurrent State Update

Equation

$$h_{t+1}=f(W h_t+Ux_t+b)$$

Plain Meaning

The next internal state depends on the current state, the new input, and learned or fixed weights.

Symbols

  • \(h_t\): current internal state.
  • \(x_t\): current input.
  • \(W,U\): weighting matrices.
  • \(b\): bias term.
  • \(f\): nonlinear update function.

How to Read It

Read it as a memory-like update rule. Past state and present input jointly shape what comes next.

Meaning in Context

The equation means the next internal state is computed from the previous state, the current input, weights, bias, and a nonlinear update rule; it is a simplified recurrent-model sketch.
