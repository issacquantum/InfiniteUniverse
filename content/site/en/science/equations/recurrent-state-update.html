Recurrent State Update

Equation

$$h_{t+1}=f(W h_t+Ux_t+b)$$

Literal Reading

Read it as: the next hidden state equals f applied to W times the current hidden state plus U times the current input plus b.

Common Reading

The next internal state depends on the current state, the new input, and learned or fixed weights.

Symbols

  • \(h_t\): current internal state.
  • \(x_t\): current input.
  • \(W,U\): weighting matrices.
  • \(b\): bias term.
  • \(f\): nonlinear update function.

Meaning of the Equation

This relation turns an algorithmic step into a precise rule. It identifies what is being updated, counted, or bounded, so the process can be checked by its state, cost, error, or learning behavior instead of only by a visual animation.
