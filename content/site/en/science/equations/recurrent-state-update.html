Recurrent State Update

Equation

$$h_{t+1}=f(W h_t+Ux_t+b)$$

Literal Reading

The next internal state is a function of the current state, the new input, and a bias term.

Model Reading

This is a simplified mathematical sketch of recurrent updating, not a full biological neuron model.

Symbol Breakdown

\(h_t\): current internal state.

\(x_t\): current input.

\(W,U\): weighting matrices.

\(b\): bias term.

\(f\): nonlinear update function.

Meaning

The equation captures memory-like dependence: the next state depends on both present input and prior state.
