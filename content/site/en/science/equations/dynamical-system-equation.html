Dynamical System Equation

Equation

$$\frac{d\mathbf{x}}{dt}=\mathbf{F}(\mathbf{x})$$

Plain Meaning

The current state determines the direction in which the system changes next.

Symbols

  • ▣\(\mathbf{x}\): state vector.
  • ▣\(t\): time.
  • ▣\(\mathbf{F}\): rule assigning change to each state.

How to Read It

Read it as a map of motion through state space. Different choices of \(\mathbf{F}\) create stability, cycles, or chaos.

What It Does Not Say

It does not guarantee easy prediction. Deterministic systems can still be chaotic.
