Dynamical System Equation

Equation

$$\frac{d\mathbf{x}}{dt}=\mathbf{F}(\mathbf{x})$$

Literal Reading

Read it as: x sub open brace t plus 1 close brace equals F open parenthesis x sub t close parenthesis.

Common Reading

The current state determines the direction in which the system changes next.

Symbols

  • \(\mathbf{x}\): state vector.
  • \(t\): time.
  • \(\mathbf{F}\): rule assigning change to each state.

Meaning of the Equation

The next state is generated from the current state; simple rules can produce complex behavior.
