Dynamical System Equation

Equation

$$\frac{d\mathbf{x}}{dt}=\mathbf{F}(\mathbf{x})$$

Literal Reading

x sub t plus 1 equals F of x sub t.

Common Reading

The current state determines the direction in which the system changes next.

Symbols

  • \(\mathbf{x}\): state vector.
  • \(t\): time.
  • \(\mathbf{F}\): rule assigning change to each state.

Meaning of the Equation

A dynamical system rule tells how the current state determines the next change. It is the general language behind motion, growth, feedback, and simulation.
