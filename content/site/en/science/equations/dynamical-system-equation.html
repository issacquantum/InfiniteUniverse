Dynamical System Equation

Equation

$$\frac{d\mathbf{x}}{dt}=\mathbf{F}(\mathbf{x})$$

Literal Reading

Read it as: the time rate of change of the state vector x equals F of x.

Common Reading

A short way to say it: The current state determines the direction in which the system changes next.

Symbols

  • \(\mathbf{x}\): state vector.
  • \(t\): time.
  • \(\mathbf{F}\): rule assigning change to each state.

Meaning of the Equation

The current state determines the direction in which the system changes next.
