Dynamic Programming Recurrence

Equation

$$ \mathrm{OPT}(i)=\min_j\{c(i,j)+\mathrm{OPT}(j)\} $$

Literal Reading

Common Reading

A hard global problem is turned into reusable local subproblems whose answers are stored instead of recomputed.

Symbols

\(\mathrm{OPT}(i)\): optimal value from state \(i\).

\(c(i,j)\): cost of moving from state \(i\) to state \(j\).

\(\min_j\): choose the best next option.

Meaning of the Equation

The method is powerful when many recursive branches ask the same subquestions. Storing those answers turns overlapping work into a table of reusable results.
