Grover Algorithm Quadratic Runtime

Equation

$$ \mathcal{O}(\sqrt{N}) $$

Literal Reading

O open parenthesis square root open parenthesis N close parenthesis close parenthesis.

Common Reading

The query complexity scaling of Grover's search algorithm over an unstructured search space of size \(N\).

Symbols

\(\mathcal{O}\): big-O notation, used to summarize asymptotic growth.

\(N\): the total number of candidate entries in the search space.

\(\sqrt{N}\): the quadratic improvement over a linear classical scan.

Meaning of the Equation

Grover search uses about the square root of N queries instead of N.
