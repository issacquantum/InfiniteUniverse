Entangled State Non-Separability

Equation

$$ |\Psi_{AB}\rangle \neq |\psi_A\rangle \otimes |\psi_B\rangle $$

Literal Reading

Psi A B is not equal to psi A tensor psi B.

Physicist's Reading

A physicist would read it as the joint state of A and B cannot be written as a product of one state for A and one state for B.

Symbol Breakdown

\(|\Psi_{AB}\rangle\): the full quantum state of the two-particle system taken together.

\(|\psi_A\rangle\): a state that would belong only to subsystem A.

\(|\psi_B\rangle\): a state that would belong only to subsystem B.

\(\otimes\): the tensor product, used to build a joint state from two independent subsystem states.

\(\neq\): not equal, meaning this factorization is impossible for an entangled state.

Physical Meaning

This is the defining mathematical feature of pure-state entanglement. If the joint state could be factorized, each subsystem would carry its own complete description. When factorization fails, only the composite system has a complete quantum description, and the subsystems must be treated as fundamentally linked.
