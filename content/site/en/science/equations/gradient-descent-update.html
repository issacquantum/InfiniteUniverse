Gradient Descent Update

Equation

$$ \theta_{t+1}=\theta_t-\eta\nabla_\theta L(\theta_t) $$

Literal Reading

The next parameter vector equals the current parameter vector minus the learning rate times the gradient of the loss.

Physicist's Reading

The model moves through parameter space in the direction that locally reduces error fastest.

Symbol Breakdown

\(\theta_t\): parameters at training step \(t\).

\(\eta\): learning rate.

\(L\): loss function being minimized.

\(\nabla_\theta L\): gradient of the loss with respect to the parameters.

Physical Meaning

Gradient descent is a controlled correction rule. It does not guarantee understanding; it only adjusts parameters to reduce a chosen objective.
