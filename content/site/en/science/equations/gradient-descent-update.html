Gradient Descent Update

Equation

$$ \theta_{t+1}=\theta_t-\eta\nabla_\theta L(\theta_t) $$

Literal Reading

Read it as: the next parameter vector equals the current parameter vector minus the learning rate times the gradient of the loss.

Common Reading

The model moves through parameter space in the direction that locally reduces error fastest.

Symbols

\(\theta_t\): parameters at training step \(t\).

\(\eta\): learning rate.

\(L\): loss function being minimized.

\(\nabla_\theta L\): gradient of the loss with respect to the parameters.

Meaning of the Equation

This relation turns an algorithmic step into a precise rule. It identifies what is being updated, counted, or bounded, so the process can be checked by its state, cost, error, or learning behavior instead of only by a visual animation.
