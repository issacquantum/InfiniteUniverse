Boltzmann Entropy

Equation

$$S_B=k_B\log W$$

Plain Meaning

Entropy grows with the number of microscopic arrangements compatible with the same large-scale description.

Symbols

  • \(S_B\): Boltzmann entropy.
  • \(k_B\): Boltzmann constant.
  • \(W\): number of compatible microstates.

How to Read It

More compatible microstates means the macrostate is statistically easier to realize.

Meaning in Context

The equation defines entropy as a count of microscopic arrangements compatible with the same visible macrostate. More compatible microstates means higher entropy.
